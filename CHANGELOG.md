## [0.1.0] — 2025-01-01

### Added

- Repository scaffold and community health files (README, LICENSE, SECURITY, CONTRIBUTING, CODE_OF_CONDUCT)
- Project specification library (`.github/specs/`)
- Docker Compose stack with Caddy, Wiki.js, and PostgreSQL
- Production environment configuration and validation
- Backup and restore workflow scripts
- Deployment workflow and production hardening
- VS Code Dev Container and recommended extensions
- Taskfile automation for common development and deployment tasks
- Google OAuth authentication and RBAC permission model
- Wiki.js custom theme and design system (typography, color palette, icons, design tokens)
- Landing page, world overview, and campaign overview content
- Branding assets (logo, favicon, social preview, README banner)
- Homepage hero and background textures
- World knowledge graph and entity relationship model
- Prompt generation pipeline architecture
- Content templates (character, NPC, location, organization, faction, session, timeline, item)
- Semantic versioning and automated release workflow
